Write Playwright automation test cases to verify the Search functionality on the File Manager page.

Test Flow:
1. Navigate to the File Manager page.
2. Identify the table containing file data.
3. Retrieve a valid file name dynamically from the table (File Name column).
4. Click on the Search input field and enter the retrieved valid file name.
5. Verify that:
   - Matching file records are displayed in the table
   - Table row count is greater than 0
6. Clear the search field.
7. Enter an incorrect or non-existent file name.
8. Verify that:
   - No records are displayed in the table
   - "No data found" or "No data shown" message is displayed
9. Clear the search field again.
10. Ensure the table resets and all records are displayed.
